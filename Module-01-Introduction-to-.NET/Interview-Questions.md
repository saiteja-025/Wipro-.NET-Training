# 🎯 Module 01 - Interview Questions

## Basic Questions

### 1. What is .NET?

**Answer:**
.NET is a software development platform developed by Microsoft for building desktop, web, mobile, cloud, and API applications.

---

### 2. Is .NET a programming language?

**Answer:**
No. .NET is a platform. C# is the programming language most commonly used with .NET.

---

### 3. What is C#?

**Answer:**
C# is an object-oriented programming language developed by Microsoft to build applications on the .NET platform.

---

### 4. Who developed .NET and C#?

**Answer:**
Microsoft.

---

## Intermediate Questions

### 5. What is the difference between .NET Framework and .NET?

| .NET Framework                          | Modern .NET        |
| --------------------------------------- | ------------------ |
| Windows only                            | Cross-platform     |
| Older platform                          | Modern platform    |
| Legacy applications                     | New applications   |
| Not actively developed for new features | Actively developed |

---

### 6. Why was .NET Core introduced?

**Answer:**
To overcome the limitations of .NET Framework by providing cross-platform support, better performance, cloud support, and open-source development.

---

### 7. Why is there no .NET Core 4?

**Answer:**
Microsoft skipped the name ".NET Core 4" to avoid confusion with ".NET Framework 4.x" and introduced .NET 5 as the unified platform.

---

### 8. What is CLR?

**Answer:**
CLR (Common Language Runtime) is the execution engine of .NET. It manages program execution, memory, exception handling, garbage collection, security, and includes the JIT compiler.

---

### 9. What is IL?

**Answer:**
IL (Intermediate Language) is the intermediate code generated by the C# compiler before being converted into machine code.

---

### 10. What is JIT?

**Answer:**
JIT (Just-In-Time Compiler) converts IL into machine code at runtime so that the CPU can execute the program.

---

## Scenario-Based Questions

### 11. Which version of .NET would you recommend for a new project in 2026?

**Answer:**
Modern .NET (currently .NET 10), because it is cross-platform, high-performance, cloud-ready, open-source, and actively developed.

---
