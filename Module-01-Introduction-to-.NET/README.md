# 📘 Module 01 - Introduction to .NET

## 📖 Overview

This module introduces the fundamentals of the .NET platform. It explains why Microsoft created .NET, how it evolved over time, the role of C#, and how a C# program is executed internally.

---

## 🎯 Learning Objectives

After completing this module, I can:

- Understand what .NET is.
- Explain why Microsoft created .NET.
- Differentiate between .NET Framework and .NET.
- Explain why .NET Core was introduced.
- Understand the role of C#.
- Explain CLR, IL, and JIT.
- Understand the execution flow of a C# program.

---

## 📚 Topics Covered

- What is .NET?
- Why Microsoft created .NET
- What is a Platform?
- .NET Framework
- .NET Core
- Evolution of .NET
- Why there is no .NET 4
- Latest .NET Version
- What is C#?
- CLR (Common Language Runtime)
- IL (Intermediate Language)
- JIT (Just-In-Time Compiler)
- C# Code Execution Flow
